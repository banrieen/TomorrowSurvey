================================================================================

灵感，每日三问，读书笔记，文集，日程。

天问，人问；天生，人生。

知乎专栏：主要以\(创业随想，市场考察\)；读书评鉴，典故阐述

在自己的网站没做起来之前：CSDN 博客相对零碎的文案，专栏系统，完整的描述。
